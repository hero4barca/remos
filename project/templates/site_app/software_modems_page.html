Expedite Series

Software Modem

The Expedite Software Modem is a software-based communication tool for controlling and communicating with small satellites.

It combines several functions into one solution: it works as a modem, controls the antenna, and schedules mission tasks. This modem can handle different types of satellite communication and formats, making it compatible with a variety of satellite systems like GomSpace and Satlab. It has a user-friendly interface, advanced diagnostic tools, and can be controlled via a REST API, allowing easy integration with other software and automation of routine operations.

Get brochure >

Is available as a web app and as well as a full modem system.

Key Features

Advanced Modem: Supports burst-mode and stream-based communications for small satellites.

Data Rate: Ranges from as low as 100 bps to 1 Mbps (10 Mbps in 2025).

Frequency: Up to 6 GHz.

Packet Formats: AX.25, all GomSpace modes, SPS-3 modes.

Automatic Doppler Correction

Line Coding: NRZ-L/M/S, Bi-Phase (Manchester/L-M/S).

Scrambler: G3RUH, CCSDS.

Transmission Procedure: Open-loop CCSDS Physical Layers Operations Procedures (PLOP-1).

External Reference: Supports 10 MHz and 1 PPS synchronization.

Advanced Frequency Sweeping: Suitable for deep-space missions.

Automatic Sequencer: Manages half-duplex communication and HPA enabling.

Automatic Pass Scheduler and Antenna Instrumentation

Support for Various Positioners: Including SPID, M2, Yaesu, Proscan, WinRotor, RotCtL.

Intuitive Desktop GUI and WebApp

REST API for Control
TCP/IP for Real-time Data Transfer
